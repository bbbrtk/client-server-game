### client-server game

to run server:

    g++ serverEpoll.cpp -o serverEpoll <port_number> 
    ./serverEpoll

to run client:

    open project (client/CMakeLists.txt) in QtCreator
    build and run

##### requirements:
Qt Creator 4.6.2

Based on Qt 5.11.1 (GCC 8.1.0, 64 bit)

cmake 3.0.0

##### made on:
Kubuntu 18.10
